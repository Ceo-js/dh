Vinculação

Vinculação

Para trabalhar com Javascript no front-end, é necessário fazer a vinculação do arquivo JS dentro do nosso arquivo HTML. Como podemos criar esse vínculo? Isso é o que você descobrirá em seguida.

Boas práticas

Uma boa prática é colocar o link com nosso arquivo js no final do corpo do nosso HTML, para que o documento já esteja carregado no momento em que o script for lido. Dessa forma, o carregamento do script não atrasa o restante da página.

Objeto window e document

Objeto window e document

Vamos ver o que cada uma dessas coisas significa: window e document de acordo com a documentação da Mozilla.

O objeto window representa a janela que contém um documento DOM; a propriedade do documento aponta para o documento DOM carregado nessa janela.

O DOM (Document Object Model, em português Modelo de Objeto do Documento) é um modelo de documento carregado no navegador da web e que representa o documento como uma árvore de nós, onde cada nó representa uma parte do documento (pode ser um elemento, uma sequência de texto ou um comentário).

Exercício 1

O subtítulo do <h3> não nos agradou e queremos adicionar um texto ao final do mesmo usando Javascript.

Para isso pedimos que, usando o método innerText selecione a tag <h3> e adicione o texto " usando código" no final.

Não se esqueça de adicionar o espaço correspondente para que os textos não fiquem grudados. Você deverá ver o seguinte resultado no seu navegador.:

// this file is read only, you cannot write it <html > <head > <script src="main.js" > </script > <title >JS Front </title > </head > <body > <h1 >Javascript Front </h1 > <h3 >Adicionando conteúdo </h3 > </body > </html > window.addEventListener('load', function() { document.querySelector( "h3" ).innerText += " usando código"; });

Exercício 2

Adicionando Tags!

Para este exercícios precisamos que você:

  • selecione a tag <body> usando querySelector()

  • e adicione a seguinte tag: <mark> Javascript ROCKS </mark>

Mas cuidado: o <body > já tem as tags <h1 > e <h3 > definidas e não queremos que elas sejam apagadas.

// this file is read only, you cannot write it <html > <head > <script src="script.js" > </script > <title >JS Front </title > </head > <body > <h1 >Javascript Front </h1 > <h3 >Adicionando conteúdo </h3 > </body > </html > // this file is read only, you cannot write it window.addEventListener( "load", function() { document.querySelector( "body" ).innerHTML += "Javascript ROCKS"; } );
