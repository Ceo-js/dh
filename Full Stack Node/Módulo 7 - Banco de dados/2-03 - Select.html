Select

Quando armazenamos informações em um banco de dados, um dos nossos objetivos futuros é poder consultá-las sempre que necessário. Para atingir este resultado, existe a diretiva select que nos permitirá, como o próprio nome indica, selecionar colunas de uma tabela para acessar as informações nela presentes.

No próximo vídeo, veremos como consultar o banco de dados através do select, ítem fundamental para poder trabalhar com bancos de dados.

 Questionário Questionário Que álbum temos? v2.0
  1. O gerente de uma gravadora musical, a MusicMundo, pediu para que organizasse o inventário de discos e um relatório com todos os álbuns (tabela albuns) que a empresa possui.

    2. Qual modelo de consulta (query) devolverá todos os álbuns?

SELECT * FROM albuns Exercício Exercício: Controle de vendas
  1. Para realizar o controle de vendas, o representante da Musicmundo pede que seja projetado um sistema que guarde o nome e telefone de todos os seus clientes. *** A tabela clientes já existe, do mesmo modo que os campos nome e telefone.

    2. Como você faria para consultar essas informações uma vez que foram gravadas na base de dados?
