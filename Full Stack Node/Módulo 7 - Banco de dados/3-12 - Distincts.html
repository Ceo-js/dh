  1. O setor de recursos humanos de MusicMundos está solicitando uma lista de todos os cargos que os funcionários têm, faça uma consulta ao nosso banco de dados que nos dá o título dos cargos que todos os funcionários têm, sem repetir registros!

Escolha a query correta

SELECT DISTINCT funcionario.cargo FROM funcionarios
