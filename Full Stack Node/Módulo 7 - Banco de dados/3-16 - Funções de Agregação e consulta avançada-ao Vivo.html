Funções de Agregação e consulta avançada [ao Vivo]

Como de costume, chegou o momento de introspecção, olhar para dentro e responder com sensatez para descobrir o quanto aprendemos sobre estes tópicos.

É por isso que antes de nos prepararmos para a aula presencial, propomos esta pequena lista de perguntas que nos permitirá saber o quão claros os conceitos em que iremos trabalhar na nossa aula presencial estão para você.

  • Quais são as três diretrizes do MySQL para inserir, atualizar e apagar registros numa tabela?

  • Com que diretriz MySQL podemos ir buscar registros em uma tabela filtrando por uma determinada condição?

  • Se fizéssemos um mecanismo de busca de página, quais são as duas coisas no MySQL que não podem faltar nas nossas consultas?

  • Para fazer uso do sistema relacional, como podemos unir as tabelas?

  • Para que serve a cláusula de filtragem DISTINCT?

  • Podemos nomear pelo menos 3 funções de agregação?

Se conseguirmos responder satisfatoriamente a todas as perguntas, estamos prontos para a aula ao vivo. Por outro lado, se achamos algumas questões difíceis, seria melhor rever o conteúdo e esclarecer estes conceitos.

Nos vemos na aula! Até lá!
