Sequelize e Configuração

Como mencionamos alguns blocos atrás, Sequelize é um ORM que nos ajuda a conectar e interagir com bancos de dados relacionais como Postgres, MySQL ou SQLite, por exemplo.

Este ORM é um pacote Node que está disponível para instalação por meio do NPM. E embora tenha uma versão um pouco mais recente, aqui trabalharemos com a versão 5 deste pacote.

Sua instalação é bastante simples, pois basta executar o comando: npm install sequelize@5.21

Além disso, você terá que configurar algumas coisas que não são todas difíceis, mas nada melhor do que um vídeo que explica todo esse processo, vamos lá então.
