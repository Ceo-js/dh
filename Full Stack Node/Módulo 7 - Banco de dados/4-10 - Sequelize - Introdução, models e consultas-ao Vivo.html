Para a aula ao vivo

Acabamos de terminar uma aula onde aprendemos muitas funcionalidades sobre esta grande ferramenta que é a Sequelize.

A aula ao vivo é uma excelente oportunidade para colocar em prática todo o conhecimento adquirido neste caso. Mas antes da reunião de Zoom, propomos, como sempre, realizar um pequeno auto questionário para saber se todos os temas foram suficientemente assimilados. Para o fazer, propomos esta lista de perguntas:

  • Além de instalar o Sequelize no projeto, que outros pacotes npm precisamos instalar?
  • Com que comando podemos gerar as pastas base (config. e models) dentro do projeto?
  • Como são definidas as colunas ou atributos de uma tabela dentro de um modelo?
  • Qual é a diferença entre os métodos findAll() e findByPk()?
  • Para que servem os operadores - como o Like- fornecidos pela Sequelize?
  • Como podemos gerar uma consulta que classifica os resultados e traz de volta um número limite de resultados?
