Sequelize - Manipulação de dados Introdução

Você lembra que ao trabalhar com MySQL existe a possibilidade de criar, atualizar e deletar registros de um banco de dados, certo?

Bem, essas mesmas tarefas também podem ser executadas no Sequelize. Mas ao contrário do MySQL, no Sequelize, para modificar um registro, você terá que usar o método update() e para excluí-lo o método destroy(). A criação permanece a mesma em ambos os casos, apenas dentro do Sequelize para criar um registro, você deve usar o método create().

Criar, atualizar ou excluir registros é uma tarefa bastante comum em sistemas de back-end e talvez, como você já sabe, esse processo é denominado CRUD, do inglês Create (Criar), Read (Ler), Update (Atualizar), Delete (Excluir).

Assim, com tudo o que foi visto aqui, ao final desta aula você poderá:

  • Implementar as tarefas de um sistema CRUD com Sequelize.
  • Entender que os métodos destroy() e update() tem, como pré-requisito, um where.
  • Saber como capturar as informações que trafegam na solicitação para serem salvas no banco de dados.
  • Implementar mecanismos de controle nos cenários em que o registro não pôde ser criado ou atualizado.

Então, vamos ver como podemos realizar essas tarefas, mas desta vez com a biblioteca Sequelize.
