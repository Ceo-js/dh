CRUD

Já falamos várias vezes sobre o que é CRUD e a importância desse processo em nosso back-end. E mais uma vez voltamos a esse conceito.
Desta vez trazemos uma sequência completa de código que busca mostrar do zero cada um dos passos necessários para podermos implementar este sistema em nossos desenvolvimentos.
Vamos preparar um bom café, lápis e papel e fazer anotações, pois o que veremos a seguir nos ajudará a aplicar os mesmos conceitos em nosso projeto integrador. Uma vez que embora este átomo de vídeo dure mais tempo do que o normal, pensamos que é um recurso inestimável em face do que você está realizando no curso.

Recomendamos anotar todos os passos necessários para criar um CRUD completo já que isso vai nos servir para realizar os exercícios.
