Raw Queries

Mesmo o Sequelize sendo bastante completo para os procedimentos do dia-a-dia, em alguns momentos, quando precisamos de operações mais específicas, acaba sendo mais fácil escrever queries literais, e é para isso que servem as chamadas raw queries.
Esta maneira mais livre de escrever SQL no código JavaScript permite que não fiquemos presos apenas nas funções pré-definidas do ORM, o que torna o Sequelize ainda mais poderoso para lidar com o banco de dados, uma vez que ele se torna flexível o bastante para se adequar à qualquer situação.
