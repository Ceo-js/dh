Com ele nós podemos testar os endpoints da nossa aplicação e analisar alguns parâmetros como, por exemplo, a duração que cada request demora para ser feito.

Axios e Node Fetch

Você se lembra do axios? Ele voltou! Axios é uma biblioteca que podemos instalar com o NPM e usá-la tanto no front-end para o navegador quanto no back-end para comunicação entre servidores. O objetivo do Axios é fazer comunicação HTTP através de código, configurando solicitações como objetos Javascript.

Existe ainda uma segunda opção para essa tarefa, que é o Node Fetch. Funciona de forma semelhante ao Fetch API do Javascript, entretanto devemos alertar que esta biblioteca, assim como o Axios, não vem de maneira padrão no NodeJs, precisando fazer a instalação através do NPM.
